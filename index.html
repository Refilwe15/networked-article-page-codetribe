KT1201: Peer-to-Peer and Client-Server Networks

Client-Server: Central server provides services and data. High security, better for large networks.

Peer-to-Peer: Devices act as both client and server. Low cost, best for small setups.

KT1202: Server Network Functionality

  • Store and share files
  • Run centralized apps
  • Manage printers and emails
  • Host intranet
  • Enable remote access (VPN)

Cloud Servers: Provide the same features with added flexibility and cost efficiency.

KT1203: Networking Devices

  • NIC: Connects computer to network
  • Repeater: Regenerates weak signals
  • Hub: Multiport repeater
  • Bridge: Connects LANs, filters by MAC
  • Switch: Multiport smart bridge
  • Router: Routes IP packets, connects networks
  • Gateway: Protocol converter
  • Brouter: Bridge + Router
